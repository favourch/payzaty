Bootstrap DropDown Effects with Animate.css

Spice up your Bootstrap dropdowns with some cool effects from Animate.css.

View Tutorial »

How to use (quick version):

Simply add the following data- attributes to your .dropdown-menu elements or their parent ul.nav element:

data-dropdown-in="EFFECT-IN-NAME" data-dropdown-out="EFFECT-OUT-NAME"

ie. data-dropdown-in="fadeIn" data-dropdown-out="fadeOut"

See the Animate.css GitHub project for a full list of effects.
NOTE: Some effects may not work as expected.
